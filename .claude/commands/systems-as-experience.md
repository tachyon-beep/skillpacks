# Systems-as-Experience Routing

**Design games where systems ARE the content, not the substrate for authored content. Orthogonal mechanics multiply into emergence; player creativity becomes gameplay; community meta-gaming extends the product. For in-engine simulation patterns (physics, AI, pathfinding, performance) use `/simulation-tactics`; for game-feel and player-experience polish use `/lyra-ux-designer`.**

Use the `using-systems-as-experience` skill from the `bravos-systems-as-experience` plugin to route to the right specialist sheet. Content authority lives in `plugins/bravos-systems-as-experience/skills/using-systems-as-experience/SKILL.md` — this wrapper is a thin pointer.

## When to Use

- Designing a sandbox, simulation, factory, exploration, or strategic-depth game
- Diagnosing why an "emergent" system isn't producing surprises
- Planning interaction matrices, feedback loops, or cascade chains
- Designing for modding, theorycrafting, or community meta-gaming
- Deciding whether a game should be systemic, authored, or hybrid

**Don't use** for: authored linear narrative (use `/creative-writing`), in-engine simulation mechanics (`/simulation-tactics`), interface and game-feel polish (`/lyra-ux-designer`), or pure game-balance math (`/yzmir-systems-thinking`).

## Sheets

### Wave 1: Foundation
- **emergent-gameplay-design** ⭐ FOUNDATIONAL — orthogonal mechanics, interaction matrices, feedback loops, cascades, systemic solutions, emergence-testing methodology

- **strategic-depth-from-systems** — orthogonal strategic axes, synergy matrices, dominant-strategy avoidance (PoE, StarCraft, Dota)

### Wave 2: Application Domains
- **optimization-as-play** — bottleneck gameplay, multi-metric optimization, satisfying production feedback (Factorio, Opus Magnum)
- **discovery-through-experimentation** — environmental hints, safe experimentation, secrets, community discovery (BotW, Outer Wilds, Noita)
- **player-driven-narratives** — systemic drama, AI storyteller, procedural characters, emergent stories (Dwarf Fortress, Rimworld, CK3)

### Wave 3: Ecosystem
- **modding-and-extensibility** — plugin architecture, mod APIs, tools, community infrastructure (Skyrim, Minecraft, Factorio)
- **community-meta-gaming** — theorycrafting, build diversity, competitive tools, persistent worlds (PoE, EVE Online, Dark Souls)
